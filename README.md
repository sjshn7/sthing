# 🔦 당신은 어떤 기묘한 이야기 캐릭터인가요?
## 📖 프로젝트 소개
넷플릭스 시리즈 **기묘한 이야기(Stranger Things)** 의 16개 캐릭터 중 나와 가장 닮은 캐릭터를 찾아주는 성격 테스트 서비스입니다.
24개의 질문에 답변하면 Claude AI가 답변을 분석해 MBTI 유형을 도출하고, 그에 맞는 캐릭터와 나의 닮은 점을 개인화된 텍스트로 보여줍니다. 결과는 카카오톡과 X(트위터)로 공유할 수 있습니다.

Claude Code를 활용해 만든 바이브 코딩 1인 프로젝트입니다.
개발 기간: 2026.05.25 ~ 2026.

## ✔ 주요 기능
- 20문항
- 16개 캐릭터 × 16개 MBTI 유형 1:1 매핑
- 카카오톡 / X(트위터) 공유 기능
- **관리자 대시보드** : 참여자 통계, 캐릭터 분포, 문항별 답변 분석
